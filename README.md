# ScreenSense: Kids Screentime Visualization

ScreenSense is a data analysis and visualization project that explores children's screen time behavior and provides simple recommendations based on usage patterns.

The project combines **data analysis, data visualization, and a web application** to make insights more interactive and practical.

---

## Project Overview

In this project, a dataset related to children's screen time was analyzed using Python.

The workflow included:

- Data cleaning and preprocessing
- Exploratory Data Analysis (EDA)
- Data visualization to identify patterns in screen usage

Python libraries such as **Pandas, NumPy, Matplotlib, and Seaborn** were used to extract meaningful insights from the dataset.

To present the insights in a more interactive way, a **Power BI dashboard** was created.

Additionally, a **web application** was developed where users can input parameters such as:

- Age
- Primary device used
- Educational screen time
- Recreational screen time
- Urban or rural background

Based on these inputs, the system generates simple recommendations related to screen usage.
