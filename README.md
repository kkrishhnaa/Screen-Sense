# ScreenSense: Kids Screentime Visualization

This project analyzes children's screen time patterns using data analysis and visualization techniques.

## Project Overview
The dataset was cleaned and analyzed using Python libraries such as Pandas, NumPy, Matplotlib, and Seaborn to extract meaningful insights.

A Power BI dashboard was created to visualize screen time trends and patterns.

Additionally, a web application was developed using React (frontend) and FastAPI (backend) where users can enter parameters such as average screen time and education-to-recreation ratio to receive a simple response about their screen usage.
